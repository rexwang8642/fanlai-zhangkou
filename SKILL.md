## 🎯 Skill 介绍
### 功能描述
「饭来张口」是一款专为懒人设计的跨平台外卖比价点单Skill，支持美团、饿了么双平台价格对比，自动计算最优优惠，直接跳转商家点单页面，解决每天点外卖「选店难、算优惠麻烦、重复操作多」的痛点。

### 适用场景
- 每天不知道吃什么，选择困难
- 想找最便宜的点单平台，不想浪费钱
- 懒得手动对比多个平台的价格和优惠
- 想快速点单，不想重复搜索和操作
- 需要定时点单，比如工作日自动点午餐

### 核心特性
| 特性 | 说明 |
|------|------|
| 双平台比价 | 同时查询美团、饿了么商家价格，自动排序推荐最优 |
| 优惠自动计算 | 自动识别所有可用优惠券、满减活动，直接显示实付价格 |
| 口味记忆 | 支持添加忌口食材，自动过滤不符合偏好的商家 |
| 历史订单记录 | 记住你常吃的菜，下次优先推荐 |
| 快速跳转 | 直接提供商家点单链接，不用再手动搜索 |
| 隐私安全 | 所有用户数据本地存储，绝不外泄 |

## 📦 安装方法
### 环境要求
- Python 3.8+
- 依赖包：requests
```bash
pip install requests
```

### 安装步骤
1. 克隆仓库：
```bash
git clone https://github.com/rexwang8642/fanlai-zhangkou.git
cd fanlai-zhangkou
```

2. 运行工具：
```bash
# 最终一键版（推荐）
python final.py 想吃的东西 [位置（可选）]

# 完整功能版
python full_version.py
```

## 🔧 配置说明
### 用户配置文件（user_config.json）
自动生成，可手动修改：
```json
{
  "disliked_foods": ["香菜", "葱", "蒜"],
  "spicy_level": "medium",
  "favorite_drink": "可乐",
  "address": "北京市朝阳区",
  "notification": "console"
}
```

### 配置项说明
| 配置项 | 说明 |
|--------|------|
| disliked_foods | 不喜欢的食材列表，搜索时自动过滤 |
| spicy_level | 辣度偏好：不辣/微辣/中辣/特辣 |
| favorite_drink | 最喜欢的饮料，推荐套餐时优先推荐 |
| address | 默认收货地址，自动定位使用 |
| notification | 通知方式：console/voice |
