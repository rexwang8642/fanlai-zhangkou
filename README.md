# 🍚「饭来张口」外卖比价点单工具
## 🎯 项目简介
「饭来张口」是一款专为懒人设计的跨平台外卖比价点单工具，支持美团、饿了么双平台价格对比，自动计算最优优惠，直接跳转商家点单页面，解决每天点外卖「选店难、算优惠麻烦、重复操作多」的痛点。

**核心亮点：**
- ✅ 双平台比价：同时查询美团、饿了么商家价格，自动排序推荐最优
- ✅ 优惠自动计算：自动识别所有可用优惠券、满减活动，直接显示实付价格
- ✅ 口味记忆：支持添加忌口食材，自动过滤不符合偏好的商家
- ✅ 历史订单记录：记住你常吃的菜，下次优先推荐
- ✅ 快速跳转：直接提供商家点单链接，不用再手动搜索
- ✅ 隐私安全：所有用户数据本地存储，绝不外泄

## 🚀 快速开始
### 环境要求
- Python 3.8+
- 依赖：`requests`
```bash
pip install requests
```

### 1. 最终一键版（推荐）
直接输入想吃的东西，自动返回最优选择和付款链接：
```bash
cd fanlai
python final.py 想吃的东西 [位置（可选）]
```
示例：
```bash
# 直接点
python final.py 麦当劳

# 指定位置
python final.py 米线 上海虹桥火车站
```

### 2. 完整功能版
进入菜单选择，支持所有功能：
```bash
python full_version.py
```
功能菜单：
1. 点单比价
2. 查看个人配置
3. 添加忌口食材
4. 查看历史订单

### 3. 快速比价版
专注比价，快速出结果：
```bash
python real_mvp.py
```

### 4. 极简版
自动显示优惠券，按预算推荐套餐：
```bash
python fanlai.py
```

## ✅ 已实现功能
### 🔍 多平台比价
- 同时查询美团、饿了么两个平台的商家数据
- 自动计算综合成本：餐品价格 + 配送费 - 最大可用优惠
- 按综合成本从低到高排序，优先推荐性价比最高的
- 同时考虑配送时间、商家评分等因素，给出综合最优选择

### 🎫 优惠自动计算
- 自动识别平台通用优惠券、商家专属优惠券、满减活动
- 计算所有可用优惠的最优组合
- 直接显示最终实付价格，不需要用户自己计算
- 支持多优惠券叠加计算，确保优惠最大化

### 🍤 口味记忆
- 用户添加的忌口食材会永久保存在配置文件中
- 搜索时自动过滤包含忌口食材的商家和餐品
- 记录用户常吃的商家和菜品，下次优先推荐
- 支持修改和删除忌口食材，灵活适配口味变化

### 📜 历史订单记录
- 自动记录用户每次的点单信息（时间、餐品、商家、价格）
- 下次点单时优先推荐最近常吃的商家和菜品
- 支持查看最近5条历史订单
- 历史数据本地存储，安全不泄露

### 🔗 快速跳转
- 每个商家都提供直接的点单链接
- 点击链接直接进入对应商家的点单页面
- 不需要用户再手动搜索商家，节省时间
- 支持美团、饿了么双平台跳转

## 🛠️ 技术方案
### 数据获取
- 放弃美团/饿了么官方API（仅对企业开放，个人无法申请）
- 采用浏览器自动化方案（Playwright/Requests）爬取公开数据
- 模拟用户正常请求，不违反平台使用规则
- 数据仅本地使用，不上传到任何服务器

### 核心算法
#### 比价算法
```
综合成本 = 餐品价格 + 配送费 - 最大可用优惠
优先级：综合成本 → 配送时间 → 商家评分
```

#### 优惠计算逻辑
1. 收集所有可用优惠券（平台券、商家券、满减）
2. 计算所有可能的优惠组合
3. 选择优惠金额最大的组合
4. 计算最终实付价格

#### 偏好过滤逻辑
1. 读取用户配置中的忌口食材列表
2. 遍历商家和餐品信息，过滤包含忌口食材的结果
3. 优先推荐用户常吃的商家和菜品
