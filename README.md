## 核心设计思路
### 1. 模块化设计
所有平台、所有功能都是独立模块，加新功能、加新平台都不用改核心代码，扩展性拉满。

### 2. 轻量优先
优先用官方API和轻量爬虫，不用重型浏览器，速度快、资源占用低、不容易被封。

### 3. 隐私优先
所有用户数据（地址、口味、支付信息）全部本地加密存储，绝不上传任何服务器，绝对安全。
